Are you a customer or an employee?

Schema

Customers

Enter a number between 100 and 104 in the customer page's text box, or id=[100-104] in the query string.

+----------------+-------------+------+-----+---------+-------+
| Field          | Type        | Null | Key | Default | Extra |
+----------------+-------------+------+-----+---------+-------+
| customerNumber | int(11)     | NO   | PRI | NULL    |       |
| firstName      | varchar(20) | YES  |     | NULL    |       |
| lastName       | varchar(20) | YES  |     | NULL    |       |
+----------------+-------------+------+-----+---------+-------+

Employees

Enter a number between 200 and 202 in the employee page's textbox, or id=[200-202] in the query string.
